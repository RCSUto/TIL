# Java TIL (기초 정리)

## 1. 개발 환경 설정
Java 프로그램을 개발하려면 JDK와 개발 도구가 필요하다.

- JDK (Java Development Kit): Java 프로그램을 컴파일하고 실행하는 도구
- IDE: IntelliJ, Eclipse 같은 개발 프로그램

### Java 실행 과정
1. .java 파일 작성
2. javac로 컴파일 → .class 파일 생성
3. java 명령어로 실행

---

## 2. Hello World
Java 프로그램의 기본 구조

ex)
```java
public class Hello {
    public static void main(String[] args) {
        System.out.println("Hello World");
    }
}
```
### 설명
+ class : Java 프로그램의 기본 단위

+ main() : 프로그램 시작 지점

+ System.out.println() : 콘솔 출력

## 3. 주석 (Comment)

## 7. 연산자
연산자는 값을 계산하거나 비교할 때 사용한다.

### 산술 연산자

+ \+ 더하기

+ \- 빼기

+ \*곱하기

+ / 나누기

ex)
```
int a = 10 + 5;
```
### 문자열 더하기
Java에서는 문자열을 +로 연결할 수 있다.

ex)
```
String name = "Tom";
System.out.println("Hello " + name);
```
### 증감 연산자
변수를 1 증가 또는 감소시킨다.

+ \++(숫자 or 변수) 1을 더한 후, 변수를 사용한다

+ \--(숫자 or 변수) 1을 뺀 후, 변수를 사용한다

+ (숫자 or 변수)\++ 변수를 사용한 후, 1을 더한다

+ (숫자 or 변수)\-- 변수를 사용한 후, 1을 뺀다

ex)
```
int a = 10;

a++; 
a--;
++a;
--a;
```
### 비교 연산자
값을 비교한다.

+ \> 크다

+ \>= 크거나 같다

+ < 작다

+ <= 작거나 같다

+ == 같다

+ != 다르다

ex)
```
int age = 20;

if (age > 18) {
    System.out.println("성인");
}
```
### 논리 연산자

조건을 결합할 때 사용한다.

+ && AND

+ || OR

+ ! NOT

ex)
```
if (age > 18 && age < 30) {
    System.out.println("20대");
}
```

## 8. 조건문
조건에 따라 다른 코드를 실행한다.

### if 문

ex)
```
int age = 20;

if (age >= 20) {
    System.out.println("성인");
} else {
    System.out.println("미성년자");
}
```
### else if
여러 조건을 처리할 때 사용

ex)
```
int score = 85;

if (score >= 90) {
    System.out.println("A");
} else if (score >= 80) {
    System.out.println("B");
} else {
    System.out.println("C");
}
```
### switch 문
여러 값을 비교할 때 사용한다.

ex)
```
int num = 2;

switch(num) {
    case 1:
        System.out.println("One");
        break;
    case 2:
        System.out.println("Two");
        break;
    default:
        System.out.println("Other");
}
```
자바 14 버전부터 새로운 switch문이 나왔다.

ex)
```
int num = 2;

switch(num) {
    case 1 -> System.out.println("One");
    case 2 -> System.out.println("Two");
    default -> System.out.println("Other");
}
```
기존의 switch문보다 더 간결해졌다.

### 삼항 연산자
간단한 조건식을 한 줄로 작성할 수 있다.

ex)
```
int age = 20;

String result = (age >= 20) ? "성인" : "미성년자";
```
