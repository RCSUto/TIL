# 인증과 인가 차이 정리

## 인증(Authentication)

사용자가 누구인지 확인하는 과정이다.

ex)
- 로그인
- 비밀번호 확인
- 지문 인증
- OTP 인증

```
아이디 입력
비밀번호 입력
```

서버가 사용자 확인

---

## 인가(Authorization)

인증된 사용자가 특정 기능을 사용할 수 있는지 확인하는 과정이다.

ex)
- 관리자 페이지 접근
- 게시글 삭제 권한
- 파일 다운로드 권한

```
ROLE_ADMIN
ROLE_USER
```

---

## 차이점

인증
-> 사용자 확인

인가
-> 권한 확인

---

## 예시 상황

로그인 성공
-> 인증 완료

관리자 페이지 접근
-> 인가 확인

---

## 정리

Authentication  
-> 누구인지 확인

Authorization  
-> 무엇을 할 수 있는지 확인
