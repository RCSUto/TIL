# Java TIL

## 12. 입력 (Scanner)
사용자로부터 값을 입력받을 때 사용한다.

### 기본 개념
- `Scanner` 클래스 사용
- 콘솔 입력 처리
ex)
```
import java.util.Scanner;

Scanner sc = new Scanner(System.in);
int num = sc.nextInt();
String str = sc.nextLine();
```

### 주요 메서드
+ nextInt() : 정수 입력

+ nextDouble() : 실수 입력

+ nextLine() : 문자열 입력

### 주의사항
+ nextInt() 이후 nextLine() 사용 시 개행 문제 발생
필요 시 nextLine() 한 번 더 호출

---

## 13. 입력 \+ 반복문 활용
입력을 반복적으로 받아 처리할 수 있다.
ex)
```
Scanner sc = new Scanner(System.in);

while(true) {
    int num = sc.nextInt();

    if(num == -1) {
        break;
    }

    System.out.println(num);
}
```
### 활용
+ 종료 조건 설정 (-1, 0 등)

+ 잘못된 입력 다시 받기

## 14. 배열 (Array)
여러 데이터를 하나로 묶어서 관리한다.

### 선언과 생성
ex)
```
int[] arr = new int[5];
```

### 특징
+ 같은 타입만 저장 가능

+ 인덱스는 0부터 시작

---

## 15. 배열 사용
ex)
```
arr[0] = 10;
System.out.println(arr[0]);
```

### 반복문과 함께 사용
ex)
```
for(int i = 0; i < arr.length; i++) {
    System.out.println(arr[i]);
}
```

---

## 16. 배열 리펙토링
중복 코드를 줄이고 가독성을 높인다
ex)
```
// before
System.out.println(arr[0]);
System.out.println(arr[1]);

// after
for(int i = 0; i < arr.length; i++) {
    System.out.println(arr[i]);
}
```

---

## 17. 2차원 배열
행과 열 구조를 가지는 배열

### 선언
ex)
```
int[][] arr = new int[3][3];
```

### 접근
ex)
```
arr[0][1] = 10;
```

### 반복문
ex)
```
for(int i = 0; i < arr.length; i++) {
    for(int j = 0; j < arr[i].length; j++) {
        System.out.println(arr[i][j]);
    }
}
```

---

## 18. 향상된 for문
배열을 더 간단하게 순회할 수 있다.
ex)
```
for(int num : arr) {
    System.out.println(num);
}
```

### 특징
+ 인덱스 사용 안함

+ 읽기 전용 반복에 적합

## 19. 메서드 (Method)
기능 단위로 코드를 나누는 구조

### 기본 구조
ex)
```
public static void hello() {
    System.out.println("Hello");
}
```

### 호출
ex)
```
hello();
```

---

## 20. 매개변수와 반환값

### 매개변수
ex)
```
public static void printNum(int num) {
    System.out.println(num);
}
```

### 반환값
ex)
```
public static int add(int a, int b) {
    return a + b;
}
```

---

## 21. 값 전달 (Call by Value)
Java는 값을 복사해서 전달한다.

ex)
```
public static void change(int x) {
    x = 100;
}
```

### 특징
+ 원본 값은 변경되지 않음

## 23. 메서드 오버로딩
같은 이름의 메서드를 여러 개 정의할 수 있다.

### 조건
+ 매개변수 타입 또는 개수가 달라야 함

ex)
```
public static int add(int a, int b) {
    return a + b;
}

public static double add(double a, double b) {
    return a + b;
}
```
