# DB (Database) 정리

## 1. DB란?
데이터를 저장하고 관리하는 시스템

+ 데이터를 체계적으로 저장

+ 빠른 조회 및 수정 가능

---

## 2. DB의 필요성
+ 데이터 중복 방지

+ 효율적인 관리

+ 안정성 확보

---

## 3. DB 종류

### 관계형 DB (RDB)
+ 테이블 형태로 데이터 저장

+ SQL 사용

ex)
+ MySQL

+ Oracle

### 비관계형 DB (NoSQL)
+ 유연한 구조

+ 대용량 데이터 처리에 적합

ex)
+ MongoDB
+ Redis

---

## 4. 기본 개념

### 테이블
데이터를 저장하는 구조

### 행(Row)
데이터 한 줄

### 열(Column)
데이터 속성

### 기본 키 (PK)
각 데이터를 구별하는 값

---

## 5. CRUD
데이터 처리 기본 작업

+ Create : 생성
+ Read : 조회
+ Update : 수정
+ Delete : 삭제

---

## 정리
DB는 데이터를 효율적으로 저장하고 관리하기 위한 핵심 시스템이며  
모든 백엔드 개발의 기본 요소이다.
